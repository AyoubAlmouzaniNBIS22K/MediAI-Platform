An intelligent healthcare platform that uses AI to analyze medical X-ray images and assist healthcare professionals in making accurate diagnoses.

## üöÄ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/YOUR_USERNAME/MediAI-Platform.git
   cd MediAI-Platform
   ```

2. **Install Python dependencies**
   ```bash
   pip install flask flask-cors pillow
   ```

3. **Run the application**
   ```bash
   python app.py
   ```

4. **Open your browser**
   Navigate to `http://localhost:5000`
